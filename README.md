# Password Strength Checker

A simple browser-based tool that checks the strength of a password in real time — no data sent anywhere, everything runs locally in your browser.

## Features

- Real-time strength feedback as you type
- Color-coded strength bar (Weak / Fair / Good / Strong)
- Checks for length, uppercase, numbers, and symbols
- Show/hide password toggle

## How to use

Just open `index.html` in any browser. No installs, no dependencies.

## Built with
